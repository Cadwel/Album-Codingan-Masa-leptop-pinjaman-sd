Essai

  1. Kelas IP Address dan masing-masing jumlah bit
    • Kelas A: Dimulai dengan bit pertama 0, memiliki jangkauan alamat 1.0.0.0 hingga 126.255.255.255. Jumlah bit untuk bagian jaringan adalah 8 bit, sedangkan untuk bagian host adalah 24 bit.
    • Kelas B: Dimulai dengan bit pertama 10, memiliki jangkauan alamat 128.0.0.0 hingga 191.255.255.255. Jumlah bit untuk bagian jaringan adalah 16 bit, sedangkan untuk bagian host adalah 16 bit.
    • Kelas C: Dimulai dengan bit pertama 110, memiliki jangkauan alamat 192.0.0.0 hingga 223.255.255.255. Jumlah bit untuk bagian jaringan adalah 24 bit, sedangkan untuk bagian host adalah 8 bit.
    • Kelas D (Multicast): Dimulai dengan bit pertama 1110, digunakan untuk mengirim multicast data. Tidak ada pembagian jaringan dan host yang khas.
    • Kelas E (Reserved): Dimulai dengan bit pertama 1111, digunakan untuk keperluan eksperimen dan penelitian. Tidak ada pembagian jaringan dan host yang khas.
  2. Fungsi IP Address:
    • Memberikan identifikasi unik kepada setiap perangkat yang terhubung dalam jaringan komputer.
    • Memungkinkan perangkat untuk saling berkomunikasi dan bertukar data melalui jaringan TCP/IP.
    • Menentukan alamat dan rute untuk pengiriman paket data antara perangkat dalam jaringan.
  3. Fungsi router, access point, kabel jaringan:
    • Router: Mengarahkan lalu lintas data antara jaringan komputer yang berbeda dan memilih jalur tercepat untuk pengiriman data. Router juga berfungsi sebagai gateway untuk menghubungkan jaringan lokal dengan internet.
    • Access Point: Menghubungkan perangkat nirkabel ke jaringan kabel, memungkinkan perangkat nirkabel seperti laptop, smartphone, atau tablet untuk terhubung ke jaringan lokal atau internet.
    • Kabel Jaringan: Menghubungkan perangkat dalam jaringan lokal secara fisik, memungkinkan transfer data antara perangkat tersebut. Jenis kabel jaringan yang umum digunakan termasuk kabel UTP (Unshielded Twisted Pair) dan kabel fiber optik.
